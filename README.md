<h1>RD_Tools</h1>
<p>this Tool is a launcher to open the RD_Module(RD-Labelfinder,AutoD365FODevSetupTool,D365FOServiceManager).

</p>
<h2>How to use:</h2>
After installation the Tool starts in the Background. 
You can open it in the Systemtray.
